Project Overview:
This project is a discrete-event simulation that models traffic flow at multiple city intersections over time.
The program simulates how vehicles move, accumulate, and clear out at intersections, 
demonstrating how traffic conditions change dynamically.

Implementation Details:
The program will use a std::map to store traffic data. The key will be the name of an intersection. 
The value will be a str::array containin three std::lists. These lists will represent:
1. Cars
2. Buses
3. Emergency vehicles
Each list will store stirng representing vehicle IDs.

The progam will read at least 100 lines of data from an external file to initialize the intersections.
Each line will include an intersection name, vehicle type, and vehicle ID,
which will be inserted into the appropriate list.

Simulation Process:
The simulation will run as a time-based system over 25 periods. During each time period: 

Vehicles may arrive at intersections and be added to a list
Vehicles may leave intersections and be removed from a list
Random events may occur, such as traffic congestion or emergency vehicle priority
